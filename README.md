# PS4-RE-TEX
PS4 RE texture creator.
# Requirements
orbis-image2gnf.exe
libSceTextureTool.dll
libSceGpuAddress.dll
libSceGnm.dll

# Usage:
Put all the requirements in the Data Folder, extract the mod with the textures, run app, wait for it to close, and you are done.
